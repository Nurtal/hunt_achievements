# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Purpose

This repository exists to "hunt" GitHub achievements — the profile badges documented at
https://github.com/drknzz/GitHub-Achievements (and shown at https://githubachievements.com/).
The goal is to perform the git/GitHub actions that unlock specific achievement badges, rather than
to build a software product.

## Current state

The repository currently contains only `README.md`. There is no source code, build system, test
suite, or package manifest yet. When asked to add code, confirm the intended language/tooling before
scaffolding, since nothing here establishes a convention.

## Working notes

- Because the aim is unlocking achievements, the *git/GitHub activity itself* (PRs, issues, commit
  co-authorship, etc.) is often the deliverable — not just the file contents. Keep this in mind when
  a request seems to be about the act of opening a PR or merging rather than the code.
- Consult the achievement list at https://github.com/drknzz/GitHub-Achievements for the exact
  conditions a given badge requires before deciding what action to take.
